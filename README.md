# iPhone 16 Pro Max - Home Screen Simulator

A faithful recreation of the iPhone 16 Pro Max home screen with the **StrongBond** social app built in.

## Features
- 📱 Realistic iPhone 16 Pro Max frame with Dynamic Island
- 🕐 Real-time clock & calendar widgets
- 📲 24 app icons with animations
- 🤝 **StrongBond App** — A Gen Z social friendship app with 3 screens:
  - Playbook: Bonding activities, card games, events, friendship wishlist
  - Streak Rewards: Unlockable perks for friendship streaks
  - Share: Instagrammable friendship streak card
